# 项目简介
基于threeJS实现的一个3D游戏场景，主要功能是玩家操控机器人探索世界，上下键操纵机器人前进，左右键调整视角，空格键跳跃。
分工：210812014杜鸣宇 场景搭建、动画实现

210812024张雨晴 环境贴图、代码整理
210812028李梦茹 模型导入、场景搭建

# 如何运行
npm install
npm run dev

# 功能特点
项目主要为键盘操控机器人移动，收集方块，下雪的粒子特效，方块和树模型的随机生成，地面使用纹理贴图绘制场景。
